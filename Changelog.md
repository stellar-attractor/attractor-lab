# Changelog — Attractor Lab

All notable architectural and structural changes to this project are documented here.

---

## [0.1.0] — 2025-03-08
### Foundation

- Repository created under the `stellar-attractor` organization.
- Project scope defined as a science-to-media engineering lab.
- Core repository structure established:
  - `core/`, `topics/`, `formats/`, `tools/`, `registry/`.
- Legal and public-facing files added:
  - README, DISCLAIMER, MIT LICENSE.
- SSH access configured for organization-level development.
- `core/` converted into a Python package (`lulab`) and installed in editable mode.
- Development workflow validated (Git, Conda, Windsurf).

This release establishes the long-term architectural foundation of the project.
